Boardborne Demo Version 1.0.0 - Grady Landers (gradxl1573@gmail.com)

The included project contains a Unity 2D game which gamifies typing by taking
inspiration from the Souls-like Action RPG genre.

Feel free to use the components of this project for academic or personal research.
DO NOT USE ANY COMPONENT OF THIS PROJECT COMMERCIALLY.

---Project Setup---
After cloning the repo from github, the game can be run natively on Windows with Boardborne.exe
in the Builds folder.

To run the game in the unity editor or modify the game with the engine, simply open this folder
from Unity Hub. The current version was created and tested using Unity Editor version 2019.4.18f1.
